El dashboard de Unified PMS ofrece una vista general del estado actual del hotel, incluyendo llegadas, salidas y ocupación. Este panel es esencial para la gestión diaria y la toma de decisiones rápidas.

## ¿Qué es y para qué sirve?

El Dashboard de Unified PMS es una herramienta centralizada que permite a los administradores de hoteles visualizar de un vistazo el rendimiento del establecimiento. Proporciona datos sobre ocupación, reservas recibidas y estado de limpieza, facilitando la planificación y gestión operativa.

![Vista general del Dashboard](../screenshots/dashboard.png)

## Exploración del Dashboard

<Steps>
  <Step title="Acceder al Dashboard" icon="mouse-pointer-click">
    Inicie sesión en el sistema y diríjase al menú lateral. Haga clic en el icono correspondiente al Dashboard para acceder a la vista principal.

    ![Acceso al Dashboard](../screenshots/dashboard.png)
  </Step>
  <Step title="Buscar reservas" icon="search">
    Utilice la barra de búsqueda en la parte superior para encontrar reservas específicas. Puede buscar por nombre, apellido, teléfono, correo o ID de reserva.

    ![Buscar reservas](../screenshots/dashboard-auto-0.png)
  </Step>
  <Step title="Revisar estado de limpieza" icon="eye">
    En la sección de limpieza, observe el número de habitaciones inspeccionadas, limpias y sucias. Esto ayuda a coordinar al personal de limpieza de manera eficiente.

    ![Estado de limpieza](../screenshots/dashboard-auto-1.png)
  </Step>
</Steps>

<Callout kind="tip">
  Use los filtros de fecha (Ayer, Hoy, Mañana) para ajustar la vista de llegadas y salidas.
</Callout>

<Callout kind="info">
  Las notificaciones push están desactivadas. Active las notificaciones para recibir alertas en tiempo real.
</Callout>

## Preguntas frecuentes

### ¿Cómo puedo activar las notificaciones push?
Para activar las notificaciones push, diríjase a la configuración de usuario y habilite las notificaciones en la sección correspondiente.

### ¿Qué información puedo buscar en la barra de búsqueda?
Puede buscar reservas utilizando el nombre, apellido, teléfono, correo electrónico o ID de reserva del huésped.
