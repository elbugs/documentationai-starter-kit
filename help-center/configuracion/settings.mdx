La sección de configuración del sistema de gestión hotelera Unified PMS ofrece un panel centralizado para gestionar diversos aspectos del hotel, desde usuarios hasta comunicaciones. Este artículo detalla cómo navegar y utilizar esta sección para optimizar la administración de su hotel.

## ¿Qué es y para qué sirve?

La sección de configuración permite a los administradores del hotel gestionar usuarios, roles, suscripciones, y más, asegurando un funcionamiento eficiente y personalizado del sistema.

<Image src="https://raw.githubusercontent.com/elbugs/hotelgest-Docs-Assets/main/screenshots/settings.png" alt="Vista principal de la página de configuración" />

## Guía paso a paso

<Steps>
  <Step title="Explorar la pestaña de usuarios" icon="users">
    En la pestaña **Usuarios**, puede administrar los usuarios de la aplicación. Aquí puede añadir nuevos usuarios o modificar los existentes.

   <Image src="https://raw.githubusercontent.com/elbugs/hotelgest-Docs-Assets/main/screenshots/settings-smart-01.png" alt="Explorar la pestaña de usuarios" />
  </Step>
  
  <Step title="Añadir un nuevo usuario" icon="plus">
    Para añadir un nuevo usuario, haga clic en **Añadir usuario**. Se abrirá un modal donde podrá ingresar los detalles del nuevo usuario.

   <Image src="https://raw.githubusercontent.com/elbugs/hotelgest-Docs-Assets/main/screenshots/settings-smart-02.png" alt="Modal para añadir usuario" />

   | Campo                | Descripción                               |
   |----------------------|-------------------------------------------|
   | Nombre               | Nombre del usuario                        |
   | Apellido             | Apellido del usuario                      |
   | Nombre de usuario    | Identificador único para el usuario       |
   | Correo electrónico   | (Opcional) Email del usuario              |
   | Contraseña           | Contraseña para el usuario                |
   | Repetir contraseña   | Confirmación de la contraseña             |
   | Rol                  | Rol asignado al usuario                   |
   | Unidades             | Unidades asignadas al usuario             |
  </Step>
  
  <Step title="Crear una API key" icon="key">
    En la sección **API keys**, puede crear nuevas claves API para integraciones. Haga clic en **Crear API key** para abrir el modal correspondiente.

   <Image src="https://raw.githubusercontent.com/elbugs/hotelgest-Docs-Assets/main/screenshots/settings-smart-07.png" alt="Modal para crear API key" />

   | Campo                | Descripción                               |
   |----------------------|-------------------------------------------|
   | Descripción de la API key | Descripción para identificar la clave |
  </Step>
  
  <Step title="Crear una plantilla de comunicación" icon="file-text">
    En la sección **Comunicación**, puede gestionar plantillas de comunicación. Haga clic en **Crear plantilla** para abrir el modal y configurar una nueva plantilla de WhatsApp.

   <Image src="https://raw.githubusercontent.com/elbugs/hotelgest-Docs-Assets/main/screenshots/settings-smart-11.png" alt="Modal para crear plantilla" />

   | Campo                | Descripción                               |
   |----------------------|-------------------------------------------|
   | Nombre               | Nombre de la plantilla                    |
   | Plantilla en Inglés  | Contenido de la plantilla en inglés       |
   | Plantilla en Español | Contenido de la plantilla en español      |
   | Plantilla en Catalán | Contenido de la plantilla en catalán      |
  </Step>
  
  <Step title="Explorar la pestaña de alojamiento" icon="bed">
    La pestaña **Alojamiento** permite gestionar las tipologías de habitaciones y sus características.

   <Image src="https://raw.githubusercontent.com/elbugs/hotelgest-Docs-Assets/main/screenshots/settings-smart-13.png" alt="Explorar la pestaña de alojamiento" />
  </Step>
  
  <Step title="Subir imágenes para tipos de habitación" icon="upload">
    En la sección de **Fotografías**, puede subir y organizar imágenes para los tipos de habitación. Haga clic en **Subir imágenes** para añadir nuevas fotos.

   <Image src="https://raw.githubusercontent.com/elbugs/hotelgest-Docs-Assets/main/screenshots/settings-smart-23.png" alt="Subir imágenes para tipos de habitación" />
  </Step>
</Steps>

## Preguntas frecuentes

<ExpandableGroup>
  <Expandable title="¿Cómo puedo desactivar las notificaciones push?" default-open="false">
    Las notificaciones push se pueden desactivar desde la configuración de notificaciones en el perfil del usuario.
  </Expandable>
  <Expandable title="¿Puedo asignar múltiples roles a un usuario?" default-open="false">
    Actualmente, cada usuario puede tener asignado un solo rol. Puede cambiar el rol desde la pestaña de usuarios.
  </Expandable>
</ExpandableGroup>
