En este artículo, exploraremos la sección de Restricciones en Unified PMS, una herramienta esencial para gestionar las reglas de estancia mínima y el cierre de ventas en el sector hotelero. Esta funcionalidad es crucial para optimizar la ocupación y maximizar los ingresos.

## ¿Qué es y para qué sirve?

La sección de Restricciones en Unified PMS permite a los administradores de hoteles establecer reglas específicas sobre la estancia mínima y el cierre de ventas. Estas configuraciones ayudan a controlar la disponibilidad de habitaciones y a gestionar las expectativas de los huéspedes de manera eficiente.

## Guía paso a paso

<Steps>
  <Step title="Acceder a la sección de Restricciones" icon="settings">
    Para comenzar, localiza el icono de ajustes en la barra lateral izquierda y selecciona la opción correspondiente a restricciones.

    <Image src="https://raw.githubusercontent.com/elbugs/hotelgest-Docs-Assets/main/screenshots/restrictions.png" alt="Vista principal de la página de restricciones" />
  </Step>
  <Step title="Crear una nueva API key" icon="key">
    Si necesitas integrar servicios externos, puedes crear una nueva API key desde el modal de creación. Haz clic en **Crear API key** y sigue las instrucciones en pantalla.

    <Image src="https://raw.githubusercontent.com/elbugs/hotelgest-Docs-Assets/main/screenshots/restrictions-smart-08.png" alt="Crear API key: Open create API key modal" />
  </Step>
  <Step title="Configurar la comunicación" icon="mail">
    Explora las opciones de comunicación para gestionar cómo se envían las notificaciones a los huéspedes. Puedes crear plantillas personalizadas para diferentes situaciones.

    <Image src="https://raw.githubusercontent.com/elbugs/hotelgest-Docs-Assets/main/screenshots/restrictions-smart-11.png" alt="Comunicación: Explore communication settings" />
  </Step>
  <Step title="Crear una plantilla de comunicación" icon="edit">
    Para crear una nueva plantilla de comunicación, abre el modal correspondiente y completa los campos necesarios. Esto te permitirá enviar mensajes personalizados a tus huéspedes.

    <Image src="https://raw.githubusercontent.com/elbugs/hotelgest-Docs-Assets/main/screenshots/restrictions-smart-12.png" alt="Crear plantilla: Open create template modal" />
  </Step>
</Steps>

## Preguntas frecuentes

<ExpandableGroup>
  <Expandable title="¿Cómo configuro una regla de estancia mínima?" default-open="false">
    Para configurar una regla de estancia mínima, accede a la sección de restricciones y selecciona la opción para añadir una nueva regla. Define el número mínimo de noches y guarda los cambios.
  </Expandable>
  <Expandable title="¿Qué sucede si cierro las ventas para una fecha específica?" default-open="false">
    Al cerrar las ventas para una fecha específica, no se podrán realizar nuevas reservas para esa fecha, lo que te permite gestionar la ocupación de manera más efectiva.
  </Expandable>
  <Expandable title="¿Puedo revertir una restricción una vez aplicada?" default-open="false">
    Sí, puedes modificar o eliminar cualquier restricción desde la misma sección donde la creaste. Simplemente selecciona la restricción y ajusta los parámetros según sea necesario.
  </Expandable>
</ExpandableGroup>
