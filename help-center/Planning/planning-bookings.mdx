El Planning de Reservas en el sistema Unified PMS es una herramienta esencial para la gestión eficiente de las reservas en un hotel. Proporciona una vista clara y organizada de todas las reservas actuales y futuras, permitiendo a los administradores y recepcionistas gestionar las operaciones diarias de manera efectiva.

## ¿Qué es y para qué sirve?

El Planning de Reservas es un calendario visual que muestra todas las reservas de un hotel. Permite a los usuarios ver y gestionar las reservas de manera intuitiva, facilitando la planificación y organización de las habitaciones y recursos del hotel.

![Vista general del Planning de Reservas](../screenshots/planning-bookings.png)

## Cómo usar el Planning de Reservas

<Steps>
  <Step title="Visualizar reservas" icon="eye">
    En la vista principal del Planning de Reservas, puede ver todas las reservas organizadas por fecha y habitación. Las reservas están codificadas por colores para facilitar su identificación.

    ![Vista principal](../screenshots/planning-bookings.png)
  </Step>
  <Step title="Hover sobre una reserva" icon="mouse-pointer-click">
    Al pasar el cursor sobre una reserva, se muestra un popover con información detallada y opciones rápidas para gestionar la reserva.

    ![Hover sobre reserva](../screenshots/planning-bookings-flow-hover-reserva---toolbar-step-0.png)
  </Step>
  <Step title="Acciones rápidas" icon="zap">
    Haga clic en el popover para acceder a acciones rápidas como abrir la reserva, añadir artículos o gestionar la limpieza.

    ![Acciones rápidas](../screenshots/planning-bookings-flow-toolbar---acciones-rápidas-step-4.png)
  </Step>
  <Step title="Abrir reserva" icon="file-text">
    Seleccione "Abrir reserva" para ver y editar detalles completos de la reserva.

    ![Abrir reserva](../screenshots/planning-bookings-flow-toolbar---acciones-rápidas-explore-4.png)
  </Step>
  <Step title="Ajustar filtros y zoom" icon="settings">
    Use las opciones de filtro para personalizar la vista del calendario, como ajustar el zoom o mostrar la ocupación.

    ![Filtros del planning](../screenshots/planning-bookings-flow-filtros-del-planning-explore-3.png)
  </Step>
  <Step title="Navegar por fechas" icon="calendar">
    Utilice las flechas en el encabezado para navegar entre diferentes fechas y ver reservas pasadas o futuras.

    ![Navegar por fechas](../screenshots/planning-bookings-auto-0.png)
  </Step>
  <Step title="Buscar reservas" icon="search">
    Use la barra de búsqueda para encontrar reservas específicas por nombre, apellido, teléfono, correo o ID de reserva.

    ![Buscar reservas](../screenshots/planning-bookings-auto-2.png)
  </Step>
</Steps>

<Callout kind="tip">
  Use el zoom para ajustar la vista del calendario según sus necesidades.
</Callout>

<Callout kind="info">
  Las reservas están codificadas por colores para indicar su estado, facilitando una rápida identificación.
</Callout>

<Callout kind="alert">
  Asegúrese de revisar las políticas de cancelación antes de realizar cambios en las reservas.
</Callout>

## Preguntas frecuentes

### ¿Cómo puedo ver todas las reservas de un mes específico?
Utilice las flechas de navegación en el encabezado del calendario para moverse entre los meses y ver todas las reservas correspondientes.

### ¿Puedo filtrar las reservas por tipo de habitación?
Sí, use las opciones de filtro para mostrar solo las reservas de ciertos tipos de habitaciones o categorías.

### ¿Cómo puedo contactar a un huésped desde el Planning de Reservas?
Al abrir una reserva, encontrará opciones de contacto directo, como llamar o enviar un mensaje por WhatsApp.

Este artículo proporciona una guía detallada sobre cómo utilizar el Planning de Reservas en el sistema Unified PMS, asegurando una gestión eficiente de las operaciones diarias del hotel.
