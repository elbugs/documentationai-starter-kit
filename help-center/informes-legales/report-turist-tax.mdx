La sección de Tasa Turística en el sistema Unified PMS permite a los administradores gestionar y revisar las tasas turísticas aplicables por Comunidades Autónomas. Esta herramienta es esencial para asegurar el cumplimiento de las normativas legales en la gestión hotelera.

## ¿Qué es y para qué sirve?

<Image src="https://raw.githubusercontent.com/elbugs/hotelgest-Docs-Assets/main/screenshots/report-turist-tax.png" alt="Vista principal de la página de Tasa Turística" />

La sección de Tasa Turística proporciona una visión clara de las tasas aplicables en diferentes regiones, facilitando la administración y el cumplimiento de las obligaciones fiscales.

## Guía paso a paso

<Steps>
  <Step title="Añadir un nuevo rol" icon="user">
    Para gestionar los permisos de usuario, puedes añadir un nuevo rol desde la sección de ajustes.

    <Image src="https://raw.githubusercontent.com/elbugs/hotelgest-Docs-Assets/main/screenshots/report-turist-tax-smart-04.png" alt="Añadir rol: Open add role modal" />
  </Step>
  <Step title="Crear una API key" icon="key">
    Las API keys son esenciales para integrar servicios externos. Aprende a crear una desde la sección de ajustes.

    <Image src="https://raw.githubusercontent.com/elbugs/hotelgest-Docs-Assets/main/screenshots/report-turist-tax-smart-08.png" alt="Crear API key: Open create API key modal" />
  </Step>
  <Step title="Configurar la comunicación" icon="mail">
    Gestiona las plantillas y preferencias de comunicación con los huéspedes desde la configuración de comunicación.

    <Image src="https://raw.githubusercontent.com/elbugs/hotelgest-Docs-Assets/main/screenshots/report-turist-tax-smart-11.png" alt="Comunicación: Explore communication settings" />
  </Step>
  <Step title="Crear una plantilla de comunicación" icon="clipboard">
    Las plantillas de comunicación permiten personalizar los mensajes enviados a los huéspedes.

    <Image src="https://raw.githubusercontent.com/elbugs/hotelgest-Docs-Assets/main/screenshots/report-turist-tax-smart-12.png" alt="Crear plantilla: Open create template modal" />
  </Step>
</Steps>

## Preguntas frecuentes

<ExpandableGroup>
  <Expandable title="¿Cómo se calculan las tasas turísticas?" default-open="false">
    Las tasas turísticas se calculan según las normativas de cada Comunidad Autónoma y pueden variar dependiendo de la temporada y el tipo de alojamiento.
  </Expandable>
  <Expandable title="¿Puedo personalizar los roles de usuario?" default-open="false">
    Sí, puedes personalizar los roles de usuario desde la sección de ajustes, permitiendo asignar permisos específicos según las necesidades del hotel.
  </Expandable>
  <Expandable title="¿Qué sucede si pierdo una API key?" default-open="false">
    Si pierdes una API key, deberás crear una nueva desde la sección de ajustes, ya que las claves no pueden ser recuperadas por motivos de seguridad.
  </Expandable>
</ExpandableGroup>
