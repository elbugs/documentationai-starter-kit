El Modelo 347 es una herramienta esencial dentro del sistema de gestión hotelera Unified PMS, diseñada para facilitar la declaración anual de operaciones con terceros. A continuación, te presentamos una guía detallada sobre cómo navegar y utilizar esta sección de manera efectiva.

## ¿Qué es y para qué sirve?

El Modelo 347 en Unified PMS permite a los hoteles gestionar y reportar las operaciones anuales con terceros, cumpliendo con las normativas legales vigentes. Esta funcionalidad es crucial para mantener la transparencia y el cumplimiento fiscal.

## Preguntas frecuentes

<ExpandableGroup>
  <Expandable title="¿Cómo puedo crear una nueva API key?" default-open="false">
    Para crear una nueva API key, dirígete a la sección de API keys y haz clic en "Crear API key". Sigue las instrucciones en pantalla para completar el proceso.
  </Expandable>

  <Expandable title="¿Qué son las políticas personalizadas?" default-open="false">
    Las políticas personalizadas son configuraciones de cancelación que puedes adaptar según las necesidades específicas de tu hotel, permitiendo una mayor flexibilidad en la gestión de reservas.
  </Expandable>

  <Expandable title="¿Cómo puedo subir imágenes para las habitaciones?" default-open="false">
    En la sección de tipologías de habitaciones, selecciona "Subir imágenes" para añadir y organizar las fotografías que se mostrarán en la web y otros canales de comunicación.
  </Expandable>
</ExpandableGroup>
