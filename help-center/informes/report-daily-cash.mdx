Caja Diaria es una sección esencial del sistema de gestión hotelera Unified PMS, diseñada para facilitar el seguimiento de los movimientos de caja. Aquí aprenderás a explorar las configuraciones de comunicación, añadir políticas de cancelación y gestionar imágenes de las tipologías de habitaciones.

## ¿Qué es y para qué sirve?

<Image src="https://raw.githubusercontent.com/elbugs/hotelgest-Docs-Assets/main/screenshots/report-daily-cash.png" alt="Vista principal de Caja Diaria" />

La sección de Caja Diaria en Unified PMS permite a los administradores de hoteles gestionar y revisar los movimientos de caja de manera eficiente. Esta herramienta es crucial para mantener un control financiero preciso y asegurar la transparencia en las operaciones diarias.

## Preguntas frecuentes

<ExpandableGroup>
  <Expandable title="¿Cómo puedo crear una nueva plantilla de comunicación?" default-open="false">
    Para crear una nueva plantilla de comunicación, dirígete a la sección de comunicación y haz clic en "Crear plantilla". Sigue las instrucciones para personalizar y guardar tu plantilla.
  </Expandable>
  <Expandable title="¿Puedo asignar más de una política de cancelación a una tipología?" default-open="false">
    No, una política no podrá estar asignada más de una vez a la misma tipología. Asegúrate de seleccionar la política adecuada para cada tipo de habitación.
  </Expandable>
  <Expandable title="¿Cómo optimizo las imágenes subidas para las habitaciones?" default-open="false">
    Al subir imágenes, el sistema genera versiones optimizadas automáticamente para asegurar una carga rápida y eficiente en la web y los correos electrónicos.
  </Expandable>
</ExpandableGroup>
