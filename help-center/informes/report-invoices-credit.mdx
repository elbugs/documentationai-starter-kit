Empieza a gestionar tus facturas rectificativas de manera eficiente con Unified PMS. Esta sección te permitirá explorar las opciones de facturación de abono y rectificación, asegurando un control preciso sobre tus informes financieros.

## ¿Qué es y para qué sirve?

La sección de **Facturas Rectificativas** en Unified PMS está diseñada para facilitar la gestión de facturas de abono y rectificación. Esto es esencial para mantener la precisión en la contabilidad y ofrecer claridad en las transacciones financieras.

## Preguntas frecuentes

<ExpandableGroup>
  <Expandable title="¿Cómo puedo añadir un nuevo rol?" default-open="false">
    Para añadir un nuevo rol, dirígete a la sección de roles y selecciona la opción de añadir. Se abrirá un modal donde podrás definir los permisos y características del nuevo rol.
  </Expandable>
  <Expandable title="¿Qué debo hacer si pierdo una API key?" default-open="false">
    Si pierdes una API key, deberás crear una nueva desde la sección de API keys. Recuerda que una vez creada, no podrás visualizarla nuevamente, así que asegúrate de guardarla en un lugar seguro.
  </Expandable>
  <Expandable title="¿Cómo configuro las preferencias de comunicación?" default-open="false">
    En la sección de comunicación, puedes ajustar las preferencias para los correos electrónicos de pre y post-estancia, así como configurar el servidor SMTP para los correos salientes.
  </Expandable>
  <Expandable title="¿Puedo personalizar las plantillas de comunicación?" default-open="false">
    Sí, puedes personalizar las plantillas de comunicación desde el modal de creación de plantillas. Aquí puedes definir el contenido en varios idiomas y utilizar etiquetas para personalizar los mensajes.
  </Expandable>
</ExpandableGroup>
