El informe contable en Unified PMS es una herramienta esencial para la gestión financiera de un hotel. Permite a los administradores acceder a datos contables detallados y realizar exportaciones, facilitando el seguimiento de las finanzas del establecimiento.

## ¿Qué es y para qué sirve?

El informe contable proporciona una vista completa de los datos financieros del hotel, permitiendo a los usuarios realizar un seguimiento preciso de las transacciones y generar informes detallados.

## Preguntas frecuentes

<ExpandableGroup>
  <Expandable title="¿Cómo puedo exportar los datos contables?" default-open="false">
    Para exportar los datos contables, dirígete a la sección de informes y selecciona la opción de exportación. Sigue las instrucciones para elegir el formato y el rango de fechas deseado.
  </Expandable>
  <Expandable title="¿Puedo personalizar los permisos de usuario?" default-open="false">
    Sí, puedes personalizar los permisos de usuario creando nuevos roles y asignándoles permisos específicos según las necesidades del hotel.
  </Expandable>
  <Expandable title="¿Cómo se configuran las plantillas de comunicación?" default-open="false">
    Las plantillas de comunicación se configuran accediendo a la sección de comunicación y seleccionando la opción de crear plantilla. Aquí puedes definir los mensajes en varios idiomas y personalizarlos con etiquetas dinámicas.
  </Expandable>
</ExpandableGroup>
