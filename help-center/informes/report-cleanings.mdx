El sistema Unified PMS ofrece una sección dedicada a la gestión de tareas de limpieza, permitiendo a los administradores de hoteles optimizar y supervisar las actividades de housekeeping. A continuación, exploraremos cómo utilizar esta sección de manera efectiva.

## ¿Qué es y para qué sirve?

La sección "Informe de Limpiezas" en Unified PMS permite a los usuarios gestionar y supervisar las tareas de limpieza de manera eficiente. Esta herramienta es esencial para asegurar que todas las habitaciones estén preparadas y listas para los huéspedes.

<Image src="https://raw.githubusercontent.com/elbugs/hotelgest-Docs-Assets/main/screenshots/report-cleanings.png" alt="Vista principal del Informe de Limpiezas" />

## Guía paso a paso

<Steps>
  <Step title="Explorar la opción de añadir un nuevo rol" icon="plus">
    Aunque no se trata directamente de la sección de limpieza, explorar la opción de añadir un nuevo rol puede ser útil para asignar tareas específicas de limpieza a diferentes usuarios.

    <Image src="https://raw.githubusercontent.com/elbugs/hotelgest-Docs-Assets/main/screenshots/report-cleanings-smart-05.png" alt="Nuevo en blanco: Añadir un nuevo rol" />
  </Step>
  <Step title="Crear una plantilla de comunicación" icon="clipboard">
    Crear plantillas de comunicación puede mejorar la coordinación con el personal de limpieza, asegurando que todos estén informados sobre las tareas y horarios.

    <Image src="https://raw.githubusercontent.com/elbugs/hotelgest-Docs-Assets/main/screenshots/report-cleanings-smart-13.png" alt="Crear plantilla de comunicación" />
  </Step>
  <Step title="Explorar la sección de alojamiento" icon="bed">
    Revisar la sección de alojamiento proporciona detalles adicionales que pueden ser relevantes para las tareas de limpieza, como la ocupación de habitaciones.

    <Image src="https://raw.githubusercontent.com/elbugs/hotelgest-Docs-Assets/main/screenshots/report-cleanings-smart-15.png" alt="Explorar la sección de alojamiento" />
  </Step>
  <Step title="Subir imágenes de tipos de habitaciones" icon="upload">
    Subir imágenes de las habitaciones ayuda a mantener un registro visual de las condiciones y necesidades de limpieza de cada tipo de habitación.

    <Image src="https://raw.githubusercontent.com/elbugs/hotelgest-Docs-Assets/main/screenshots/report-cleanings-smart-24.png" alt="Subir imágenes de tipos de habitaciones" />
  </Step>
  <Step title="Desplazarse para ver más opciones" icon="eye">
    Desplazarse por la sección de tipos de habitaciones permite ver más contenido y opciones relevantes para la gestión de limpieza.

    <Image src="https://raw.githubusercontent.com/elbugs/hotelgest-Docs-Assets/main/screenshots/report-cleanings-smart-26.png" alt="Desplazarse para ver más contenido" />
  </Step>
</Steps>

## Preguntas frecuentes

<ExpandableGroup>
  <Expandable title="¿Cómo puedo asignar tareas de limpieza a un usuario específico?" default-open="false">
    Para asignar tareas de limpieza a un usuario específico, primero asegúrese de que el usuario tenga el rol adecuado. Luego, utilice la sección de roles para asignar las tareas correspondientes.
  </Expandable>
  <Expandable title="¿Es posible personalizar las plantillas de comunicación?" default-open="false">
    Sí, las plantillas de comunicación se pueden personalizar para incluir información específica sobre las tareas de limpieza y horarios.
  </Expandable>
  <Expandable title="¿Cómo puedo subir imágenes de las habitaciones?" default-open="false">
    Para subir imágenes, diríjase a la sección de tipos de habitaciones y utilice la opción "Subir imágenes" para agregar fotos relevantes.
  </Expandable>
</ExpandableGroup>
