El informe de Wallet en el sistema Unified PMS permite a los usuarios gestionar el saldo y los movimientos financieros de manera eficiente. Esta sección es esencial para mantener un control detallado de las transacciones dentro del sistema de gestión hotelera.

## ¿Qué es y para qué sirve?

El informe de Wallet proporciona una visión clara del saldo actual y los movimientos financieros asociados a las operaciones del hotel. Es una herramienta crucial para la administración financiera, permitiendo a los usuarios revisar y auditar transacciones de manera efectiva.

![Vista general del informe de Wallet](../screenshots/report-wallet.png)

## Cómo usar el informe de Wallet

<Steps>
  <Step title="Acceder al informe de Wallet" icon="mouse-pointer-click">
    Navegue al menú lateral izquierdo y haga clic en el icono correspondiente al informe de Wallet.

    ![Acceder al informe](../screenshots/report-wallet.png)
  </Step>
  <Step title="Revisar el saldo y movimientos" icon="eye">
    Una vez dentro, podrá ver el saldo actual y los movimientos detallados. Asegúrese de revisar cada transacción para mantener un control adecuado.

    ![Revisar saldo y movimientos](../screenshots/report-wallet.png)
  </Step>
</Steps>

<Callout kind="tip">
  Mantenga un registro regular de los movimientos para evitar discrepancias financieras.
</Callout>

<Callout kind="info">
  Esta sección no muestra tablas ni formularios, pero proporciona una visión clara del estado financiero.
</Callout>

## Preguntas frecuentes

### ¿Cómo puedo ver los detalles de una transacción específica?
Actualmente, el informe de Wallet muestra un resumen de las transacciones. Para detalles específicos, consulte la sección de transacciones en el sistema principal.

### ¿Puedo exportar el informe de Wallet?
El sistema no ofrece una opción de exportación directa desde esta vista. Considere utilizar herramientas de captura de pantalla o consultar con el soporte técnico para opciones avanzadas.

### ¿Qué hacer si detecto un error en el saldo?
Revise los movimientos recientes para identificar posibles errores. Si el problema persiste, contacte al soporte técnico para asistencia adicional.
