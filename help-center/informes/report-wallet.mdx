El informe de Wallet en Unified PMS permite a los administradores de hoteles gestionar eficientemente el saldo y los movimientos financieros. Esta sección proporciona una visión clara y detallada de las transacciones, facilitando el control financiero del establecimiento.

## ¿Qué es y para qué sirve?

<Image src="https://raw.githubusercontent.com/elbugs/hotelgest-Docs-Assets/main/screenshots/report-wallet.png" alt="Vista principal del Informe de Wallet" />

El informe de Wallet es una herramienta esencial para visualizar y administrar el saldo y los movimientos financieros dentro del sistema de gestión hotelera. Permite a los usuarios monitorear las transacciones de manera eficiente, asegurando un control preciso de las finanzas del hotel.
