En este artículo, exploraremos la sección de "Informe de Régimenes" en el sistema Unified PMS. Esta sección es esencial para gestionar los servicios de pensión ofrecidos por el hotel. A continuación, te guiaremos a través de los pasos para navegar y utilizar esta funcionalidad de manera efectiva.

## ¿Qué es y para qué sirve?

La sección "Informe de Régimenes" en Unified PMS permite a los administradores de hoteles gestionar y supervisar los servicios de pensión. Esta herramienta es crucial para optimizar la oferta de servicios y mejorar la experiencia del huésped.

<Image src="https://raw.githubusercontent.com/elbugs/hotelgest-Docs-Assets/main/screenshots/report-boards.png" alt="Vista principal de la página de Informe de Régimenes" />

## Guía paso a paso

<Steps>
  <Step title="Acceder a la sección de Roles" icon="users">
    Para gestionar los roles de usuario, dirígete a la sección de **Roles** en el menú de ajustes.

    <Image src="https://raw.githubusercontent.com/elbugs/hotelgest-Docs-Assets/main/screenshots/report-boards-smart-04.png" alt="Añadir rol: Open add role modal" />
  </Step>
  <Step title="Crear una API Key" icon="key">
    En la sección de **API keys**, puedes crear nuevas claves de API para integrar servicios externos.

    <Image src="https://raw.githubusercontent.com/elbugs/hotelgest-Docs-Assets/main/screenshots/report-boards-smart-09.png" alt="Crear API key: Open create API key modal" />
  </Step>
  <Step title="Configurar la comunicación" icon="mail">
    Gestiona las configuraciones de comunicación para personalizar la interacción con los huéspedes.

    <Image src="https://raw.githubusercontent.com/elbugs/hotelgest-Docs-Assets/main/screenshots/report-boards-smart-12.png" alt="Comunicación: Explore communication settings" />
  </Step>
  <Step title="Crear una plantilla de comunicación" icon="edit">
    Puedes crear plantillas de comunicación para WhatsApp y otros medios, asegurando mensajes consistentes.

    <Image src="https://raw.githubusercontent.com/elbugs/hotelgest-Docs-Assets/main/screenshots/report-boards-smart-13.png" alt="Crear plantilla: Open create communication template modal" />
  </Step>
</Steps>

## Preguntas frecuentes

<ExpandableGroup>
  <Expandable title="¿Cómo puedo añadir un nuevo rol?" default-open="false">
    Para añadir un nuevo rol, ve a la sección de **Roles** en el menú de ajustes y selecciona **Añadir rol**. Sigue las instrucciones en pantalla para completar el proceso.
  </Expandable>
  <Expandable title="¿Qué debo hacer si pierdo una API Key?" default-open="false">
    Si pierdes una API Key, deberás crear una nueva desde la sección de **API keys**. Recuerda que una vez creada, no podrás visualizarla nuevamente, así que asegúrate de guardarla en un lugar seguro.
  </Expandable>
  <Expandable title="¿Cómo configuro el servidor SMTP?" default-open="false">
    En la sección de **Comunicación**, puedes configurar el servidor SMTP para gestionar los correos electrónicos de confirmación. Asegúrate de ingresar correctamente los detalles del servidor para evitar problemas de envío.
  </Expandable>
</ExpandableGroup>
